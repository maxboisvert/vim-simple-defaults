# Vim simple defaults
This is a light distribution to get started with vim.

## Insatallation
With Vim-plug :

```
call plug#begin()
Plug 'maxboisvert/vim-simple-defaults'
call plug#end()

silent! source $HOME/.vim/plugged/vim-simple-defaults/defaults.vim
```
