# Vim simple defaults
This is a light distribution to get started with vim with sane defaults.

## Insatallation
With Vim-plug (or your prefered way) :
```
call plug#begin()
Plug 'maxboisvert/vim-simple-defaults'
call plug#end()

" let g:vsd_tab_length = 4
runtime! plugin/vim-simple-defaults.vim

" Your vim configuration here
" ...
```
